NAME
    base64 - turn any data into safe text, or turn that text back

WHAT IT DOES
    base64 takes any file and turns it into a string of letters, numbers,
    and symbols that are safe to put in emails, web pages, or anywhere
    that only handles regular text. It can also go the other way and turn
    that safe text back into the original data.

    Think of it like a secret code where every piece of data gets turned
    into regular letters so it can travel safely.

HOW TO USE IT
    base64 [options] [FILE]
    base64 --decode [options] [FILE]

    When no FILE is given, base64 reads from the keyboard (standard input).

OPTIONS
    -d, --decode
        Go the other way: take base64 text and turn it back into the
        original data.

    -i, --ignore-garbage
        When decoding, skip over any characters that are not part of
        base64. Useful when the input has extra spaces or line endings.

    -w N, --wrap=N
        Wrap the output at N characters per line (default is 76).
        Use -w 0 to turn off wrapping entirely.

    --help
        Show a short help message and quit.

EXAMPLES
    base64 photo.jpg > photo.txt
        Turn a photo into base64 text saved in photo.txt.

    base64 --decode photo.txt > photo.jpg
        Turn the base64 text back into the original photo.

    echo "hello" | base64
        Turn the word "hello" into base64 right away.

    echo "aGVsbG8K" | base64 --decode
        Turn base64 text back into the original word.
