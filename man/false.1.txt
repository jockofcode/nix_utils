NAME
    false - do nothing and report failure

WHAT IT DOES
    false does absolutely nothing and then exits with a failure code (1).
    This tells the shell that something went wrong, even though nothing
    actually happened.

    This sounds useless, but it is helpful in shell scripts when you
    need to force a failure condition or test how your script handles
    errors.

    See also true(1), which does nothing but reports success.

HOW TO USE IT
    false

OPTIONS
    false ignores all arguments and options, including --help.

EXAMPLES
    false
        Does nothing, exits with status 1 (failure).

    false && echo "This never prints"
        Since false reports failure, the echo never runs.

    false || echo "This always prints"
        Since false reports failure, the echo always runs.

    while false; do
      echo "This loop never runs"
    done
        A loop that never executes.

SEE ALSO
    true(1)
