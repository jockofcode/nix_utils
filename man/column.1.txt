NAME
    column - line up text into neat columns

WHAT IT DOES
    column takes lines of text and arranges them into tidy columns, like
    a table. This makes lists of information much easier to read.

    For example, a messy list of names and ages separated by commas can be
    turned into a clean table where everything lines up perfectly.

HOW TO USE IT
    column [options] [FILE]...

    When no FILE is given, column reads from standard input.

OPTIONS
    -t, --table
        Table mode. Treats each line as a row and lines up all the
        fields into aligned columns.

    -s SEP, --separator=SEP, --input-separator=SEP
        Use SEP as the character that separates fields.
        Default is any whitespace (spaces or tabs).

    -o SEP, --output-separator=SEP
        Use SEP between columns in the output. Default is two spaces.

    -c N, --output-width=N
        Try to fit the output into N characters wide when not in
        table mode.

    -x, --fillrows
        Fill columns across instead of down when displaying a list.

    --help
        Show a short help message and quit.

EXAMPLES
    column -t data.txt
        Neatly align the fields in data.txt as a table.

    echo -e "name,age\nalice,10\nbob,11" | column -t -s ","
        Turn comma-separated data into a table:
          name   age
          alice  10
          bob    11

    ls | column -c 80
        List files arranged in columns fitting 80 characters wide.
