NAME
    env - show environment variables or run a command with changed settings

WHAT IT DOES
    Environment variables are like named settings that programs can read.
    For example, HOME tells programs where your home folder is.

    env either shows you all current environment variables, or runs a
    program with some variables temporarily changed or added.

HOW TO USE IT
    env [options] [NAME=VALUE]... [COMMAND [ARGS]...]

OPTIONS
    -i, --ignore-environment
        Start with no environment variables at all, then only use the
        ones you provide.

    -u NAME, --unset=NAME
        Remove NAME from the environment before running the command.

    -0, --null
        End each variable with a zero byte instead of a newline when
        printing.

    --help
        Show a short help message and quit.

EXAMPLES
    env
        Print all current environment variables.

    env HOME=/tmp myprogram
        Run myprogram with HOME set to /tmp.

    env -i PATH=/usr/bin myprogram
        Run myprogram with a completely clean environment, only PATH set.

    env -u DISPLAY myprogram
        Run myprogram without the DISPLAY variable.
