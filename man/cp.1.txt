NAME
    cp - copy files and directories

WHAT IT DOES
    cp makes a copy of a file or directory. The original stays where it is
    and a new copy appears at the place you choose. You can copy one file
    to a new name, or copy many files into a folder.

HOW TO USE IT
    cp [options] SOURCE DEST
    cp [options] SOURCE... DIRECTORY

OPTIONS
    -r, -R, --recursive
        Copy directories and everything inside them.

    -i, --interactive
        Ask before overwriting a file that already exists.

    -f, --force
        If the destination cannot be opened, remove it and try again.

    -n, --no-clobber
        Never overwrite a file that already exists.

    -u, --update
        Only copy if the source is newer than the destination, or the
        destination does not exist yet.

    -v, --verbose
        Print the name of each file as it is copied.

    -p, --preserve
        Keep the original file's timestamps and permissions.

    -a, --archive
        Same as -rp: copy recursively and preserve everything.

    -l, --link
        Make hard links instead of copying the file data.

    -s, --symbolic-link
        Make symbolic links instead of copying the file data.

    --help
        Show a short help message and quit.

EXAMPLES
    cp notes.txt notes_backup.txt
        Copy notes.txt to a new file called notes_backup.txt.

    cp photo.jpg /backup/
        Copy photo.jpg into the /backup/ folder.

    cp -r projects/ projects_backup/
        Copy the entire projects folder and its contents.

    cp -v *.txt /docs/
        Copy all .txt files to /docs/ and show each one as it copies.
