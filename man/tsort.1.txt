NAME
    tsort - sort a list where some items must come before others

WHAT IT DOES
    tsort (topological sort) solves ordering problems. You give it a
    list of pairs, where each pair means "this item must come before
    that item." tsort figures out an order that satisfies all the rules.

    Think of it like planning a schedule where some tasks must be done
    before others. For example: "put on socks before shoes" and
    "put on shirt before jacket."

    If the rules are impossible (two things each must come before the
    other), tsort reports a cycle and prints what it can.

HOW TO USE IT
    tsort [FILE]

    The file should contain pairs of items, one pair per line or
    space-separated. When no FILE is given, tsort reads from standard input.

EXAMPLES
    echo -e "socks shoes\nshirt jacket\nunderwear pants\npants shoes" | tsort
        Prints an order that respects all the rules:
          socks
          underwear
          shirt
          pants
          shoes
          jacket

    tsort dependencies.txt
        Sort a list of software build dependencies.

NOTES
    tsort is commonly used to determine the order in which to build
    programs when some programs depend on others being built first.

    When a cycle is detected (A must come before B, and B must come
    before A), tsort prints a warning to standard error and outputs
    the cycle members anyway.
