NAME
    pwd - print the current working directory

WHAT IT DOES
    pwd (print working directory) tells you which folder you are
    currently in. When you open a terminal, you start somewhere in
    the file system, and pwd shows you exactly where that is.

HOW TO USE IT
    pwd [options]

OPTIONS
    -L, --logical
        Print the path as it appears logically, following any
        symbolic links as if they were real folders. This is the default.

    -P, --physical
        Print the real path with all symbolic links resolved to show
        the actual location on disk.

    --help
        Show a short help message and quit.

EXAMPLES
    pwd
        Print the current directory.
        Example output: /home/alice/projects

    pwd -P
        Print the real path even if you navigated through symlinks.
