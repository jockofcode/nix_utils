NAME
    rm - delete files and directories

WHAT IT DOES
    rm deletes files. Once deleted with rm, files do not go to a trash
    can — they are gone right away. Be careful!

    By default rm only deletes files, not folders. To delete folders and
    everything inside them, use -r.

HOW TO USE IT
    rm [options] FILE...

OPTIONS
    -r, -R, --recursive
        Delete a directory and everything inside it. This is needed to
        remove any non-empty folder.

    -f, --force
        Never ask questions and ignore errors about files that do not
        exist. Useful in scripts.

    -i
        Ask "are you sure?" before deleting each file.

    -I
        Ask once before deleting more than 3 files or when using -r.
        Less noisy than -i but still gives you a safety check.

    -v, --verbose
        Print the name of each file as it is deleted.

    --help
        Show a short help message and quit.

EXAMPLES
    rm old_notes.txt
        Delete the file old_notes.txt.

    rm -r temp_folder/
        Delete temp_folder/ and everything inside it.

    rm -i *.log
        Delete each .log file but ask before each one.

    rm -rf build/
        Force-delete the build/ folder without asking anything.
        Warning: there is no undo!
