LOGNAME(1)                       User Commands                       LOGNAME(1)

NAME
    logname - print your login name

WHAT IT DOES
    logname prints the name you used when you logged into the computer.
    Unlike whoami, which shows the current effective user, logname always
    shows the original login name — even if you have switched to a
    different user using su or sudo.

HOW TO USE IT
    logname

OPTIONS
    --help
        Show a short help message and quit.

EXAMPLES
    logname
        Prints your login name, for example: alice

SEE ALSO
    whoami(1), id(1)
