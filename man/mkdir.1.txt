NAME
    mkdir - create a new folder

WHAT IT DOES
    mkdir (make directory) creates one or more new empty folders.
    If the folder already exists, mkdir reports an error — unless you
    use the -p option, which lets it silently skip existing folders and
    also create any needed parent folders.

HOW TO USE IT
    mkdir [options] DIRECTORY...

OPTIONS
    -p, --parents
        Create the folder and any missing parent folders along the way.
        Also, do not report an error if the folder already exists.
        Example: mkdir -p a/b/c creates a, then a/b, then a/b/c.

    -m MODE, --mode=MODE
        Set the permissions for the new folder. MODE is a number like
        755 or 700. See chmod(1) for details on permission modes.

    -v, --verbose
        Print a message for each folder that is created.

    --help
        Show a short help message and quit.

EXAMPLES
    mkdir photos
        Create a folder called photos.

    mkdir -p projects/ruby/myapp
        Create the whole path at once, even if some parts do not exist.

    mkdir -m 700 private
        Create a private folder that only you can read.

    mkdir -v docs logs backup
        Create three folders and print their names as they are made.

SEE ALSO
    rmdir(1), rm(1)
