NAME
    whoami - print your current username

WHAT IT DOES
    whoami prints the username of the person currently running the
    program. It is the simplest way to find out what user you are.

    It shows the effective user — the one whose permissions you have
    right now. If you used sudo to become root, whoami shows "root."

HOW TO USE IT
    whoami

OPTIONS
    --help
        Show a short help message and quit.

EXAMPLES
    whoami
        Prints your username, for example: alice

    sudo whoami
        Prints: root

SEE ALSO
    id(1), logname(1)
