NAME
    sha224sum - calculate or check SHA-224 fingerprints for files

WHAT IT DOES
    sha224sum creates a 56-character SHA-224 fingerprint (checksum) for
    each file. SHA-224 is a variant of SHA-256 that produces a shorter
    output. It is part of the SHA-2 family of secure hash algorithms.

    This fingerprint is like a unique ID for the file's contents —
    if the file changes even slightly, the fingerprint will be
    completely different.

HOW TO USE IT
    sha224sum [options] [FILE]...

    When no FILE is given, sha224sum reads from standard input.

OPTIONS
    -b, --binary
        Read the file in binary mode (marks output with *).

    -c, --check
        Read fingerprints from FILE and check that the files still match.

    --tag
        Print results in BSD style: SHA224 (filename) = fingerprint.

    --quiet
        When checking, only report files that FAILED.

    --status
        When checking, do not print anything; just use the exit code.

    -w, --warn
        When checking, warn about badly formatted checksum lines.

    --help
        Show a short help message and quit.

EXAMPLES
    sha224sum document.pdf
        Print the SHA-224 fingerprint for document.pdf.

    sha224sum -c checksums.txt
        Verify that files match their saved fingerprints.

SEE ALSO
    sha256sum(1), sha512sum(1), md5sum(1)
