NAME
    tac - print a file in reverse order (last line first)

WHAT IT DOES
    tac is cat spelled backwards, and it does the opposite: it prints
    the lines of a file starting from the last line and ending with the
    first line. The content of each line stays the same — only the order
    of lines is reversed.

    This is useful when you want to see the most recent entries in a log
    file first, or when another tool expects input in reverse order.

HOW TO USE IT
    tac [options] [FILE]...

    When no FILE is given, tac reads from standard input.

OPTIONS
    -s SEP, --separator=SEP
        Use SEP as the record separator instead of a newline. Records
        ending with SEP are reversed instead of newline-terminated lines.

    -r, --regex
        Treat SEP as a regular expression pattern.

    -b, --before
        The separator comes before each record instead of after.

    --help
        Show a short help message and quit.

EXAMPLES
    tac log.txt
        Print the last line of log.txt first, then work backward.

    tac log.txt | head -5
        Show the last 5 lines of a log file.

    seq 1 5 | tac
        Count backward: 5  4  3  2  1

    cat(1), rev(1)
