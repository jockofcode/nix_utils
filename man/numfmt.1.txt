NAME
    numfmt - convert numbers to and from human-readable form

WHAT IT DOES
    numfmt reformats numbers on lines of text, converting between raw
    numbers and human-friendly sizes. For example, it can convert
    1073741824 to 1.0G, or convert 1.5K to 1500.

    This is useful for making numbers in files or command output easier
    to read, or for converting human-readable sizes back to raw numbers
    for calculations.

HOW TO USE IT
    numfmt [options] [NUMBER]...

    When no NUMBER is given, numfmt reads lines from standard input.

OPTIONS
    --from=UNIT
        Treat the input numbers as already scaled by this unit:
          none   plain numbers (default)
          auto   auto-detect K, M, G etc.
          si     1K=1000, 1M=1000000, etc.
          iec    1K=1024, 1M=1048576, etc.
          iec-i  same as iec but uses Ki, Mi notation

    --to=UNIT
        Scale the output to this unit (same choices as --from).
        none prints the raw number.

    --field=N
        Which field (column) of each input line to convert.
        Default is the first field.

    --padding=N
        Pad the output to at least N characters wide.

    --round=METHOD
        How to round when scaling: up, down, from-zero, towards-zero,
        or nearest (default).

    --suffix=SUFFIX
        Append SUFFIX to each formatted number.

    --help
        Show a short help message and quit.

EXAMPLES
    numfmt --to=si 1000000
        Prints: 1.0M

    numfmt --to=iec 1073741824
        Prints: 1.0G

    numfmt --from=si 1.5k
        Prints: 1500

    du -b | numfmt --from=none --to=si
        Show disk usage in human-readable sizes.
