NAME
    head - show the first few lines of a file

WHAT IT DOES
    head prints the first 10 lines of a file (or a number of lines you
    choose). This is great for taking a quick peek at the beginning of
    a file without printing the whole thing.

    When given more than one file, head shows a header before each one
    so you know which file each section came from.

HOW TO USE IT
    head [options] [FILE]...

    When no FILE is given, head reads from standard input.

OPTIONS
    -n N, --lines=N
        Show the first N lines instead of the default 10.
        Use -n -N (with a minus sign) to print everything
        except the last N lines.

    -c N, --bytes=N
        Show the first N bytes instead of lines.

    -q, --quiet, --silent
        Do not print the file name headers when reading multiple files.

    -v, --verbose
        Always print the file name header, even for just one file.

    --help
        Show a short help message and quit.

EXAMPLES
    head notes.txt
        Print the first 10 lines of notes.txt.

    head -n 5 notes.txt
        Print only the first 5 lines.

    head -n 20 file1.txt file2.txt
        Print the first 20 lines of each file with headers.

    head -c 100 data.bin
        Print the first 100 bytes.
