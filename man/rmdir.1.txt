NAME
    rmdir - remove an empty directory

WHAT IT DOES
    rmdir deletes empty directories (folders). If the folder still has
    any files or other folders inside it, rmdir will refuse to delete it
    and show an error. To remove a folder that has things inside it, use
    rm -r instead.

HOW TO USE IT
    rmdir [options] DIRECTORY...

OPTIONS
    -p, --parents
        After removing the directory, also try to remove its parent
        directories if they are now empty too. Useful for cleaning up
        a whole path at once.

    -v, --verbose
        Print a message for each directory that is removed.

    --help
        Show a short help message and quit.

EXAMPLES
    rmdir empty_folder/
        Remove the empty folder called empty_folder/.

    rmdir -p a/b/c
        Remove c, then b (if empty), then a (if empty).

    rmdir -v logs/ cache/ tmp/
        Remove all three empty folders and print their names.

SEE ALSO
    rm(1), mkdir(1)
