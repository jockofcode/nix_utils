NAME
    unexpand - convert spaces back into tab characters

WHAT IT DOES
    unexpand is the opposite of expand. It takes spaces in a file and
    converts them back into tab characters where possible. By default
    it only converts leading spaces (the indentation at the start of
    each line). Use -a to also convert spaces inside lines.

    This is useful when you want to reduce file size or when a program
    requires tabs for indentation.

HOW TO USE IT
    unexpand [options] [FILE]...

    When no FILE is given, unexpand reads from standard input.

OPTIONS
    -a, --all
        Convert all groups of spaces throughout each line, not just
        the leading whitespace.

    -t N, --tabs=N
        Set tab stops every N columns. Default is 8.
        Spaces that align to a tab stop become a tab character.

    -t LIST
        Set tab stops at specific column positions, separated by commas.

    --help
        Show a short help message and quit.

EXAMPLES
    unexpand spaced.txt
        Convert leading spaces to tabs using 8-column tab stops.

    unexpand -a -t 4 code.rb
        Convert all groups of 4 spaces to tabs throughout the file.

    unexpand code.rb > code_tabs.rb
        Save the tab-converted version to a new file.
