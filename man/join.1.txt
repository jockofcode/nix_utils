NAME
    join - combine two files by matching a shared field

WHAT IT DOES
    join looks at two files and finds lines that have the same value in
    a chosen field (column). When they match, it merges them into one
    combined line.

    Think of it like combining two lists. If one list has student names
    and IDs, and another has student IDs and grades, join can combine
    them into one list with names, IDs, and grades all together.

    Both files must be sorted on the join field before you run join.

HOW TO USE IT
    join [options] FILE1 FILE2

    Use - as a filename to read from standard input.

OPTIONS
    -j N
        Join on field number N in both files (default is field 1).

    -1 N
        Join on field N in FILE1.

    -2 N
        Join on field N in FILE2.

    -t CHAR
        Use CHAR as the field separator (default is any whitespace).

    -o FORMAT
        Control what fields appear in the output.
        Example: -o 1.1,2.2 shows field 1 from FILE1 and field 2 from FILE2.

    -a N
        Also print lines from file N that had no matching line in the
        other file (like a left or right join in databases).

    -v N
        Print only the lines from file N that had no match (no joined
        output at all).

    -i, --ignore-case
        Ignore upper/lower case differences when comparing join fields.

    --help
        Show a short help message and quit.

EXAMPLES
    join names.txt grades.txt
        Combine lines from both files that share the same first field.

    join -t "," -1 2 -2 1 file1.csv file2.csv
        Join comma-separated files on field 2 of file1 and field 1 of file2.

    join -a 1 names.txt grades.txt
        Show all lines from names.txt, including those with no match.
