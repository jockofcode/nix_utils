NAME
    shuf - randomly shuffle lines

WHAT IT DOES
    shuf takes a list of lines and mixes them up in a random order,
    then prints the shuffled result. This is like shuffling a deck of
    cards — all the same lines come out, just in a random order.

    You can also use shuf to pick a random sample from a list.

HOW TO USE IT
    shuf [options] [FILE]
    shuf -e [options] [ARG]...
    shuf -i LO-HI [options]

    When no FILE is given, shuf reads from standard input.

OPTIONS
    -e, --echo
        Treat each argument as a line to shuffle, instead of reading
        from a file.

    -i LO-HI, --input-range=LO-HI
        Generate and shuffle numbers from LO to HI.

    -n N, --head-count=N
        Only output N lines from the shuffled result.
        Useful for picking a random sample.

    -r, --repeat
        Lines can be picked more than once. Use with -n to draw N
        samples with replacement.

    -o FILE, --output=FILE
        Write the output to FILE instead of the screen.

    --help
        Show a short help message and quit.

EXAMPLES
    shuf names.txt
        Print the names in a random order.

    shuf -n 1 choices.txt
        Pick one random line from the file.

    shuf -e apple banana cherry
        Pick a random order for these three items.

    shuf -i 1-100 -n 5
        Pick 5 random numbers between 1 and 100.
