NAME
    mv - move or rename a file

WHAT IT DOES
    mv moves a file from one place to another, or renames it. If you
    move a file to a new name in the same folder, it is renamed. If you
    move it to a different folder, it is relocated. The original file
    disappears from its old location.

HOW TO USE IT
    mv [options] SOURCE DEST
    mv [options] SOURCE... DIRECTORY

OPTIONS
    -i, --interactive
        Ask before overwriting a file that already exists.

    -f, --force
        Do not ask before overwriting. Silently overwrite existing files.

    -n, --no-clobber
        Never overwrite a file that already exists. The move is skipped.

    -u, --update
        Only move if SOURCE is newer than DEST, or DEST does not exist.

    -v, --verbose
        Print the name of each file as it is moved.

    --help
        Show a short help message and quit.

EXAMPLES
    mv old_name.txt new_name.txt
        Rename a file in the same folder.

    mv report.txt /backup/
        Move report.txt into the /backup/ folder.

    mv -i notes.txt /shared/
        Move notes.txt but ask before overwriting if a file exists there.

    mv -v *.log /var/logs/
        Move all .log files and print each one as it moves.
