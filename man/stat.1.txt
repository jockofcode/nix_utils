NAME
    stat - show detailed information about a file

WHAT IT DOES
    stat displays everything the operating system knows about a file:
    its size, when it was created and last modified, who owns it, what
    permissions it has, and much more. It is like a very detailed version
    of the information you see with ls -l.

HOW TO USE IT
    stat [options] FILE...

OPTIONS
    -f, --file-system
        Show information about the file system the file is on,
        not the file itself.

    -c FORMAT, --format=FORMAT
        Print only the fields listed in FORMAT. Useful for scripts.
        Common format codes:
          %n  file name
          %s  size in bytes
          %U  owner's user name
          %G  owner's group name
          %A  permissions in human form (like -rwxr-xr-x)
          %a  permissions as an octal number (like 755)
          %x  last access time
          %y  last modification time
          %z  last status change time
          %i  inode number
          %h  number of hard links

    -t, --terse
        Print all information on one short line. Good for scripts.

    --help
        Show a short help message and quit.

EXAMPLES
    stat report.txt
        Show all information about report.txt.

    stat -c "%n: %s bytes" *.txt
        Print the name and size of each .txt file.

    stat -c "%A %U %n" files/
        Print permissions, owner, and name for each file.

    stat -f /home
        Show information about the file system containing /home.
