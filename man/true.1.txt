NAME
    true - do nothing and report success

WHAT IT DOES
    true does absolutely nothing and then exits with a success code (0).
    This tells the shell that everything went fine, even though nothing
    actually happened.

    This sounds useless, but it is helpful in shell scripts when you
    need to force a success condition, make a loop run forever, or
    fill in a placeholder command.

    See also false(1), which does nothing but reports failure.

HOW TO USE IT
    true

OPTIONS
    true ignores all arguments and options, including --help.

EXAMPLES
    true
        Does nothing, exits with status 0 (success).

    true && echo "This always prints"
        Since true reports success, the echo always runs.

    true || echo "This never prints"
        Since true reports success, the echo never runs.

    while true; do
      echo "This loops forever"
      sleep 1
    done
        A loop that runs until you press Ctrl+C.

SEE ALSO
    false(1)
