NAME
    nproc - print how many processor cores are available

WHAT IT DOES
    nproc tells you how many processor cores your computer can use for
    running programs. Programs that can work on many things at once
    (in parallel) use this number to decide how many tasks to run
    at the same time.

    By default it shows how many cores are available to the current
    process, which may be less than the total if the system has limited
    your access.

HOW TO USE IT
    nproc [options]

OPTIONS
    --all
        Show the total number of processors installed in the computer,
        not just the ones currently available to you.

    --ignore=N
        Subtract N from the result, but never go below 1.
        This leaves N cores free for other things.

    --help
        Show a short help message and quit.

EXAMPLES
    nproc
        Print the number of available processor cores.
        Example output: 8

    nproc --all
        Print the total number of installed processors.

    nproc --ignore=2
        Print the number of cores minus 2, useful to avoid using
        all cores and freezing the computer.

    make -j$(nproc)
        Compile a program using all available cores.
