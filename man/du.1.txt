NAME
    du - find out how much disk space files and folders are using

WHAT IT DOES
    du (disk usage) tells you how much storage space a file or folder
    takes up on your disk. If you give it a folder, it will check the
    folder and everything inside it. This helps you find out which files
    or folders are taking up the most space.

HOW TO USE IT
    du [options] [FILE]...

    When no FILE is given, du checks the current directory.

OPTIONS
    -h, --human-readable
        Show sizes in easy-to-read form like 4.0K, 2.5M, or 1.1G
        instead of raw numbers.

    -s, --summarize
        Show only the total for each argument, not each sub-folder.

    -a, --all
        Show sizes for individual files too, not just directories.

    -c, --total
        After all results, print a grand total.

    -d N, --max-depth=N
        Only go N levels deep into sub-folders. -d 1 shows only the
        immediate contents of a folder.

    -k
        Show sizes in kilobytes (1024 bytes each).

    -m
        Show sizes in megabytes.

    --si
        Use 1000-based sizes (kB, MB, GB) instead of 1024-based.

    -b, --bytes
        Show the exact number of bytes each file takes.

    --help
        Show a short help message and quit.

EXAMPLES
    du -h documents/
        Show how much space the documents folder uses, in readable form.

    du -sh downloads/
        Show just the total size of the downloads folder.

    du -h -d 1 /home/alice/
        Show sizes of each folder one level deep in /home/alice/.

    du -ah | sort -h
        Show all files and folders sorted from smallest to largest.
