NAME
    yes - repeatedly print the same text over and over

WHAT IT DOES
    yes prints the same text over and over again, one line at a time,
    forever (until you stop it with Ctrl+C or until another program
    stops reading from it). By default it prints "y" over and over.

    This is useful for automatically answering "yes" to programs that
    keep asking you a question. You pipe yes into the other program
    so it gets "y" as the answer every time it asks.

HOW TO USE IT
    yes [STRING]...

    When no STRING is given, yes prints "y" repeatedly.
    When you give strings, it prints them joined by spaces, repeatedly.

OPTIONS
    yes has no options (other than --help, which it ignores).

EXAMPLES
    yes
        Prints "y" forever. Press Ctrl+C to stop.

    yes | some_program
        Answer "y" to every question some_program asks.

    yes "I agree"
        Prints "I agree" over and over forever.

    yes n | some_program
        Answer "n" to every question some_program asks.

    yes "" | head -5
        Print 5 empty lines.
