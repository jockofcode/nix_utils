NAME
    printf - format and print text with precise control

WHAT IT DOES
    printf prints text using a format string that lets you control exactly
    how the output looks. Unlike echo, printf does not add a newline
    automatically and gives you more options for formatting numbers,
    aligning text, and inserting special characters.

    The format string contains regular text and special codes starting
    with % or backslash.

HOW TO USE IT
    printf FORMAT [ARGUMENT]...

FORMAT CODES
    %s    Insert the next argument as a string.
    %d    Insert the next argument as a whole number (integer).
    %f    Insert the next argument as a decimal number.
    %o    Insert the next argument as an octal number.
    %x    Insert the next argument as a hex number.
    %%    A literal percent sign.

    Between % and the letter you can add:
      -    left-align in the field
      N    minimum field width of N characters
      .N   for %f, N decimal places

ESCAPE SEQUENCES
    \n    newline
    \t    tab
    \\    backslash
    \a    bell

    --help
        Show a short help message and quit.

EXAMPLES
    printf "Hello, %s!\n" Alice
        Prints: Hello, Alice!

    printf "%d + %d = %d\n" 3 4 7
        Prints: 3 + 4 = 7

    printf "%-10s %5d\n" apple 42
        Prints: apple          42
        (left-aligned name, right-aligned number)

    printf "%.2f\n" 3.14159
        Prints: 3.14
