NAME
    fold - wrap long lines to fit a certain width

WHAT IT DOES
    fold takes lines of text that are too long and breaks them up so
    each piece fits within a set number of characters (default: 80).
    Unlike fmt, fold wraps every line individually without caring about
    paragraphs or words — it just cuts at the exact column limit.

    This is useful when you need text that fits in a fixed-width display.

HOW TO USE IT
    fold [options] [FILE]...

    When no FILE is given, fold reads from standard input.

OPTIONS
    -w N, --width=N
        Wrap at N characters wide. Default is 80.

    -s, --spaces
        Instead of cutting in the middle of a word, try to break at
        the last space before the limit. This makes output more readable.

    -b, --bytes
        Count bytes instead of characters when measuring width.

    --help
        Show a short help message and quit.

EXAMPLES
    fold -w 60 long_lines.txt
        Wrap each line to 60 characters.

    fold -s -w 72 email.txt
        Wrap at 72 characters, breaking at spaces instead of
        cutting through words.

    echo "This is a very long line that needs to be wrapped" | fold -w 20
        Prints:
          This is a very long
           line that needs to
           be wrapped
