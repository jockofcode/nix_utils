NAME
    touch - update a file's timestamp or create an empty file

WHAT IT DOES
    touch does two things: if a file already exists, it updates the
    "last modified" and "last accessed" time stamps to right now. If the
    file does not exist, touch creates a new empty file.

    The most common use is to quickly create empty files, or to tell the
    system that a file has been "touched" so that tools which check
    timestamps (like make) know to rebuild it.

HOW TO USE IT
    touch [options] FILE...

OPTIONS
    -a
        Update only the last access time, not the modification time.

    -m
        Update only the modification time, not the access time.

    -c, --no-create
        Do not create the file if it does not exist; just skip it.

    -t STAMP
        Use STAMP as the time instead of right now.
        Format: [[CC]YY]MMDDhhmm[.ss]
        Example: 202601011200.00 means January 1, 2026, noon.

    -d STRING, --date=STRING
        Use a date described in plain text, like "2026-01-01 12:00".

    -r FILE, --reference=FILE
        Copy the timestamps from FILE instead of using the current time.

    --help
        Show a short help message and quit.

EXAMPLES
    touch newfile.txt
        Create an empty file called newfile.txt, or update its time
        if it already exists.

    touch -c missing.txt
        Do nothing if missing.txt does not exist.

    touch -t 202601011200 file.txt
        Set the timestamp to January 1, 2026 at noon.

    touch -r original.txt copy.txt
        Give copy.txt the same timestamps as original.txt.
