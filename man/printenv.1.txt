NAME
    printenv - print the value of environment variables

WHAT IT DOES
    printenv shows you the values of environment variables. Environment
    variables are named settings that programs can read. For example,
    HOME holds the path to your home folder, and PATH lists the folders
    the computer searches for programs.

    If you give printenv a variable name, it prints just that variable's
    value. With no arguments, it prints all variables.

HOW TO USE IT
    printenv [options] [VARIABLE]...

OPTIONS
    -0, --null
        End each output line with a zero byte instead of a newline.
        This helps other programs that handle values containing newlines.

    --help
        Show a short help message and quit.

EXIT STATUS
    If any named variable is not set, printenv exits with status 1.
    Otherwise it exits with 0.

EXAMPLES
    printenv
        Print all environment variables (sorted alphabetically).

    printenv HOME
        Print the value of HOME, for example: /home/alice

    printenv PATH
        Print the list of program search directories.

    printenv HOME USER
        Print the values of HOME and USER on separate lines.
