NAME
    basename - get just the file name from a full path

WHAT IT DOES
    When you have a long path like /home/alice/documents/report.txt,
    basename gives you just the last part: report.txt. It can also remove
    a file ending (like .txt) if you tell it to.

    Think of it like cutting off the address part of a mailing label so
    you only see the person's name.

HOW TO USE IT
    basename NAME [SUFFIX]
    basename [options] NAME...

OPTIONS
    SUFFIX
        If the name ends with SUFFIX, remove it too.
        Example: basename report.txt .txt gives you report.

    -a, --multiple
        Allow more than one name at a time (one per line of output).

    -s SUFFIX, --suffix=SUFFIX
        Remove this ending from each name. Use this with -a when
        processing multiple names.

    -z, --zero
        End each output line with a zero byte instead of a newline.
        This helps other programs that handle file names with spaces.

    --help
        Show a short help message and quit.

EXAMPLES
    basename /home/alice/report.txt
        Prints: report.txt

    basename /home/alice/report.txt .txt
        Prints: report

    basename -a /foo/bar /baz/qux
        Prints:
          bar
          qux

    basename -a -s .txt /docs/a.txt /docs/b.txt
        Prints:
          a
          b
