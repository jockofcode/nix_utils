NAME
    cat - print one or more files to the screen

WHAT IT DOES
    cat reads files one after another and prints everything to the screen.
    The name comes from "concatenate," which means to join things end to end.
    You can also use it to see what is inside a short file quickly, or to
    glue several files together into one big output.

    When no file is given (or when a file is named "-"), cat reads whatever
    you type at the keyboard.

HOW TO USE IT
    cat [options] [FILE]...

OPTIONS
    -n, --number
        Put a number at the start of every line.

    -b, --number-nonblank
        Put a number at the start of every line that is not empty.
        This overrides -n.

    -s, --squeeze-blank
        When there are several blank lines in a row, collapse them
        into just one blank line.

    -E, --show-ends
        Show a dollar sign ($) at the end of each line so you can
        see exactly where lines end.

    -T, --show-tabs
        Show tab characters as ^I so you can see where tabs are.

    -A, --show-all
        Same as turning on -E and -T at the same time.

    -e
        Same as -E (show line endings).

    -t
        Same as -T (show tabs).

    -v, --show-nonprinting
        Show invisible control characters using ^ and M- notation.

    --help
        Show a short help message and quit.

EXAMPLES
    cat notes.txt
        Print the contents of notes.txt.

    cat file1.txt file2.txt
        Print file1.txt then file2.txt one after the other.

    cat -n story.txt
        Print story.txt with line numbers.

    cat -s messy.txt
        Print messy.txt but squish repeated blank lines into one.

    cat file1.txt file2.txt > combined.txt
        Join both files and save the result to combined.txt.
