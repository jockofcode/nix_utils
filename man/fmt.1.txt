NAME
    fmt - reformat text paragraphs to a target line width

WHAT IT DOES
    fmt takes paragraphs of text and reformats them so each line is
    not too long and not too short. It joins short lines together and
    wraps long lines to fit within the target width (default: 75
    characters). Blank lines separate paragraphs and are kept as-is.

    This is useful for cleaning up text files or emails where the lines
    are too long or poorly wrapped.

HOW TO USE IT
    fmt [options] [FILE]...

    When no FILE is given, fmt reads from standard input.

OPTIONS
    -w N, --width=N
        Wrap lines at N characters wide. Default is 75.

    -s, --split-only
        Only split long lines; do not join short lines together.

    -u, --uniform-spacing
        Put exactly one space between words (two after a period).

    -p PREFIX, --prefix=PREFIX
        Only reformat lines that begin with PREFIX, keeping PREFIX
        on each line.

    --help
        Show a short help message and quit.

EXAMPLES
    fmt long_email.txt
        Reformat the email to 75 characters per line.

    fmt -w 60 notes.txt
        Reformat notes.txt to fit in 60 characters.

    fmt -s wide.txt
        Only split lines that are too long; leave short ones alone.

    cat messy.txt | fmt > clean.txt
        Reformat and save to a new file.
