NAME
    realpath - print the full, real path of a file

WHAT IT DOES
    realpath takes a file path and prints the complete, absolute path
    with all symlinks resolved and all . and .. pieces cleaned up.
    You always get the true, full location of the file.

    This is useful in scripts where you need to know the exact location
    of a file no matter how it was referred to.

HOW TO USE IT
    realpath [options] FILE...

OPTIONS
    -e, --canonicalize-existing
        Print an error if any part of the path does not exist.

    -m, --canonicalize-missing
        Do not check whether the path exists; just clean it up.

    -s, --strip, --no-symlinks
        Do not resolve symbolic links; only clean up . and .. parts.

    -q, --quiet
        Do not print errors for non-existent paths.

    -z, --zero
        End each output line with a zero byte instead of a newline.

    --relative-to=DIR
        Print the path relative to DIR instead of an absolute path.

    --relative-base=DIR
        Print a relative path only if it falls within DIR; otherwise
        print an absolute path.

    --help
        Show a short help message and quit.

EXAMPLES
    realpath ../myfile.txt
        Print the absolute path from a relative one.
        Example output: /home/alice/myfile.txt

    realpath -e /etc/resolv.conf
        Print the real path, failing if the file does not exist.

    realpath /usr/bin/python
        Follow symlinks and print where Python actually lives.
