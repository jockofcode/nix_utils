NAME
    tee - send output to both the screen and a file at the same time

WHAT IT DOES
    tee copies everything it reads from standard input to two places
    at once: the screen (standard output) and one or more files. The
    name comes from a T-shaped pipe fitting that splits a flow in two.

    This is useful when you want to watch a program's output live but
    also save it to a file for later.

HOW TO USE IT
    tee [options] [FILE]...

    When no FILE is given, tee just passes the input through to the screen.

OPTIONS
    -a, --append
        Add to the end of each file instead of overwriting it.

    -i, --ignore-interrupts
        Ignore the interrupt signal (Ctrl+C). Useful in pipelines.

    --help
        Show a short help message and quit.

EXAMPLES
    ls -la | tee listing.txt
        Show the directory listing on screen and save it to listing.txt.

    ./build.sh | tee build.log
        Watch the build output and save it to a log file.

    echo "new entry" | tee -a log.txt
        Add a line to log.txt without erasing what was already there.

    ls | tee file1.txt file2.txt
        Send the output to two files at the same time.
