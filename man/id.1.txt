NAME
    id - show who you are logged in as

WHAT IT DOES
    id prints information about your user account on the computer.
    Every user has a user ID number (UID) and belongs to one or more
    groups that have group ID numbers (GID). id shows all of these.

    This is useful for checking your permissions or understanding why
    you can or cannot access certain files.

HOW TO USE IT
    id [options] [USER]

    When no USER is given, id shows information about you (the current user).

OPTIONS
    -u, --user
        Print only your user ID number.

    -g, --group
        Print only your primary group ID number.

    -G, --groups
        Print all group ID numbers you belong to.

    -n, --name
        Print the name instead of the number. Use with -u, -g, or -G.

    -r, --real
        Print the real ID instead of the effective ID. Use with -u or -g.

    -z, --zero
        Separate output with zero bytes instead of spaces or newlines.

    --help
        Show a short help message and quit.

EXAMPLES
    id
        Print all information about yourself.
        Example output: uid=1000(alice) gid=1000(alice) groups=1000(alice),4(adm)

    id -u
        Print just your user ID number.

    id -un
        Print just your username.

    id bob
        Print information about the user named bob.
