NAME
    sed - find and replace text in a stream

WHAT IT DOES
    sed (stream editor) reads lines of text, applies editing commands to
    each line, and prints the result. The most common use is the
    substitute command s, which finds a pattern and replaces it with
    something else — like a find-and-replace tool that works on files
    from the command line.

    sed reads each line, applies your commands, and outputs the result.
    It does not change the original file unless you use -i.

HOW TO USE IT
    sed [options] SCRIPT [FILE]...
    sed [options] -e SCRIPT [-e SCRIPT]... [FILE]...
    sed [options] -f SCRIPT_FILE [FILE]...

    When no FILE is given, sed reads from standard input.

COMMON COMMANDS
    s/PATTERN/REPLACEMENT/
        Replace the first match of PATTERN with REPLACEMENT on each line.

    s/PATTERN/REPLACEMENT/g
        Replace all matches on each line (global replace).

    s/PATTERN/REPLACEMENT/i
        Replace ignoring upper/lower case differences.

    d   Delete lines that match the address.

    p   Print the current line (usually used with -n to print only
        matching lines).

    q   Quit after processing the current line.

OPTIONS
    -n, --quiet, --silent
        Do not automatically print each line. Only print when you use
        the p command explicitly.

    -e SCRIPT, --expression=SCRIPT
        Add a command to run. Use multiple -e options for several commands.

    -f FILE, --file=FILE
        Read commands from FILE instead of from the command line.

    -i[SUFFIX], --in-place[=SUFFIX]
        Edit the file in place (modify it directly). If SUFFIX is given,
        save a backup copy with that suffix first.

    --help
        Show a short help message and quit.

EXAMPLES
    sed 's/cat/dog/' animals.txt
        Replace the first "cat" on each line with "dog."

    sed 's/cat/dog/g' animals.txt
        Replace every "cat" on each line with "dog."

    sed -n '/error/p' log.txt
        Print only lines containing "error."

    sed '/^#/d' config.txt
        Delete all lines starting with #.

    sed -i 's/foo/bar/g' myfile.txt
        Replace "foo" with "bar" directly in myfile.txt.

    sed -e 's/a/A/' -e 's/b/B/' text.txt
        Apply two substitutions in one pass.
