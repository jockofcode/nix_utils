NAME
    wc - count lines, words, and characters in a file

WHAT IT DOES
    wc (word count) counts things in files: how many lines, how many
    words, and how many bytes (or characters). By default it shows all
    three counts. With options, you can pick just the one you want.

    When given multiple files, wc also prints a total line at the end.

HOW TO USE IT
    wc [options] [FILE]...

    When no FILE is given, wc reads from standard input.

OPTIONS
    -l, --lines
        Count only lines.

    -w, --words
        Count only words (sequences of characters separated by spaces).

    -c, --bytes
        Count only bytes.

    -m, --chars
        Count only characters (same as bytes for ASCII text, but
        different for multi-byte characters like emojis).

    -L, --max-line-length
        Print the length of the longest line in the file.

    --help
        Show a short help message and quit.

EXAMPLES
    wc essay.txt
        Print the line count, word count, and byte count.
        Example output:  42  356 2048 essay.txt

    wc -l log.txt
        Count only the number of lines.

    wc -w story.txt
        Count only the number of words.

    wc -c photo.jpg
        Show the file size in bytes.

    wc *.txt
        Count all three totals for every .txt file, plus a grand total.

    cat file.txt | wc -l
        Count lines piped from another command.
