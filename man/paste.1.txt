NAME
    paste - combine lines from multiple files side by side

WHAT IT DOES
    paste takes two or more files and puts their lines side by side,
    joined by a tab character. The first line of each file becomes one
    combined line, then the second line of each file, and so on.

    Think of it like taking several columns of a spreadsheet and placing
    them next to each other.

HOW TO USE IT
    paste [options] [FILE]...

    Use - as a filename to include a column from standard input.
    When no FILE is given, paste reads from standard input.

OPTIONS
    -d LIST, --delimiters=LIST
        Use the characters in LIST as separators instead of tabs.
        If LIST has more than one character, they are used in rotation.
        Example: -d "," uses commas.

    -s, --serial
        Instead of combining files in parallel, paste all lines from
        each file into one long line, then move to the next file.

    -z, --zero-terminated
        Use a zero byte to end lines instead of a newline.

    --help
        Show a short help message and quit.

EXAMPLES
    paste names.txt ages.txt
        Combine the two files side by side with tabs:
          alice    10
          bob      11

    paste -d "," names.txt ages.txt
        Use commas as separators:
          alice,10
          bob,11

    paste -s numbers.txt
        Combine all lines of numbers.txt into one line:
          1    2    3    4    5

    paste - - < data.txt
        Read stdin and combine every two lines side by side.
